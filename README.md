# Ionization-and-Photocurrent-model

Numerical modeling of strong-field ionization and photocurrent generation in a solid-state medium using Python.

## Overview

This repository contains two small computational physics scripts focused on ultrafast laser–matter interaction:

- **Ionization rate vs intensity** based on the **Keldysh formalism**
- **Photocurrent vs time** with an **avalanche mechanism** included

The project is built as a compact numerical experiment: define physical constants and pulse parameters, evaluate the model on a grid, and visualize the resulting dependencies.
